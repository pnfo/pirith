මා මං හිංසි දිපාදකො = මා දීපාදක සත්වයෙක් නො පෙළාවා
මා මං හිංසි චතුස්පද‍ො = මා සිවුපා සත්වයෙක් නො පෙළාවා
මා මං හිංසි බහුප්පද‍ො = මා බොහෝ පා ඇති සත්වයෙක් නො පෙළාවා

සබෙ සත්තා = සියලු සත්හු ද
සබ්බ පාණා = සියලු ප්‍රාණීහුද
සබ්බෙ භුතා ච කෙවලා = හුදෙක් සියලු භූතයෝ ද
සබ්බෙ = හැමදෙන
භද්‍රානි = යහපතක්ම
පස්සන්තු = දකිත්වා.
මා කඤ්චි පාපමාගමා = කිසිවකුත් කරා දුකෙක් නො පැමිණේවා!

අප්පමාණො බුද්ධො = බුද්ධ ගුණය අපමණය
අප්පමාණො ධම්මො = දහම් ගුණය අපමණය
අප්පමාණො සංඝො = සංඝ ගුණය අපමණය

පමාණවන්තානි = ප්‍රමාණ කළ හැක
සිරිංසපානි = සිරිංසපයෝ හෙවත් දීර්ඝ ජාතීහු (එනම්)
අභි = සර්පයෝ ය
විච්ඡිකා = ගෝනුස්සෝ ය
සතපදී = සතපදීහු ය (පත්තෑ ආදීහු)
උණ්ණානාභි = මකුළුවෝ ය
සරබු = සිකනල්ලු ය
මූසිකා = මූසිකය හ (යන මේ වශයෙනි)
කතා මෙ රක්ඛා = මා විසින් ආරක්ෂා කරන ලද්ද‍ාහ
කතා මෙ පරිත්තා = මා විසින් පිරිත් කරන ලද්ද‍ාහ
පටික්කමන්තු = පහවෙත්වා
භූතානි = භූතයෝ
සො අහං = ඒ මම
නමො = නමස්කාර කරමි
භගවතො = භාග්‍යවතුන් වහන්සේට
නමො = නමස්කාර කරමි
සත්තන්නං සම්මා සම්බුද්ධානං = සත් සම්බුදුවරයන්ට වහන්සේ ලාට
